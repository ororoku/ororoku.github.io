# 資産運用の勉強

## 不動産投資
例えば、都内に新築で5000万円のマンションを買い、月17万円で貸すケースについて考えてみる。


### メリット
* 長期的な資金繰り
  * 将来長期間にわたって安定的なキャッシュフローが期待できる
  * 老後に政府の年金支給が滞った場合の備え
* 資産価格の上昇期待
  * 都市部の不動産価格は世界中で上昇傾向にあるため、日本でもまだまだ上がるだろうというもの
* 節税
  * 最初の数年間は給与所得を圧縮して所得税の還付を受けられる
 
### デメリット
以下の様なリスクがある。
* 市場リスク
  * 不動産の値下がり、バブルの崩壊
  * 価格が半分になれば、最大3000万円程度の損失の恐れがある
* 流動性リスク
  * 売りたい時に売れない可能性
* 空き部屋リスク
  * 借り手がつかない場合
  * 最悪、自分で住めば良いが、、
  * 都内なら恐らく借り手には困らない？
* 家賃の見直し
  * 物件の経年に伴い、入居者が入れ替わるタイミングで家賃が下がっていくのが普通
  * ローンに家賃保証が付いている場合も、たいてい保証家賃の金額は2年ごとに見直される

### 試算
前提
* 利回り
  * 年間の収入(17*12 = 204)/試算価値 なので、4%程度
  * 10年間で3割価格が下落する場合、キャピタルロスは3%程度
  * したがって実質利回りは1%程度、諸費用を考えるとほぼプラマイゼロか
* 所有に伴う費用
  * 年間管理費 : 数万円
  * 年間修繕積立金 : 数万円
  * 年間固定資産税 : 数万円
  * 借り手がいない時の広告費用 
* 売買に伴う費用
  * 登記など各種手数料 : 数十万円
  * 仲介手数料 : 物件価格の3% ~ 150万円

シナリオ
* 好況と値上がりが続いた場合
* すぐにバブルが崩壊した場合

### 考察
最大のリスクである不動産価格の下落の可能性をどう考えれば良いのだろうか。
現在の価格がバブルであるということは、様々な市場の指標が示している（要出典）。
売り抜けるタイミングについてだが、価格が下がり始めた時にすぐに売り抜け出来なかった場合が怖い。
