# 資産運用の勉強

## 不動産投資
例えば、都内に新築で5000万円のマンションを買い、月17万円で貸すケースについて考えてみる。

### メリット
* 老後の資金繰り
  * 将来にわたって安定的なキャッシュフローが期待できる
  * 政府の年金支給が不履行になった場合の備え
* 資産価格の上昇期待
  * 都市部の不動産価格は、世界中で上昇傾向にある
  * 
 
### デメリット
以下の様なリスクがある。
* 市場リスク
  * 不動産の値下がり、バブルの崩壊
  * 価格が半分になれば、最大3000万円程度の損失の恐れがある
* 流動性リスク
  * 売りたい時に売れない可能性
* 空き部屋リスク
  * 借り手がつかない場合
  * 最悪、自分で住めば良いが、、
  * 都内なら恐らく借り手には困らない？

### 考察
結局、最大のリスクである不動産価格の下落をどう考えれば良いのだろうか。
現在の価格がバブルであるということは、様々な市場の指標が示している（要出典）。
