# 経済学の勉強の記録

私の元々のバックグラウンドは物理です。

素粒子・加速器実験で博士号取得後、経済学の大学院に入り直すことを思い立ち、一から経済学を勉強し始めてもうすぐ3年が経とうとしております。

3年間は決して順風満帆なものではありませんでしたが、どうにかこうにか、2つ目の修士号を経済学でいただけることになりそうです。

理系だった頃の知り合いに再会すると、経済学ってどうやって勉強すれば良いの？と聞かれることがあります。私が聞きたいくらいですが、純粋に理系の世界に生きていた時は、経済学に触れる機会も、経済学部の友人もほとんど皆無だったことを思い出すと、私の経験を書いておくことも何かの参考にはなるのかもしれない、と思いました。

## 大学院（再）入学まで
まず、2013年4月、経済学の知識がほぼゼロ（駒場の教養の授業で受けたことがあった程度）の状態で、最初に手に取った経済学の本は遠山智久さんの『大学生のミクロ経済学』でした。この本を読みながら、練習問題を一通り解きました。夏に経済学研究科の入試を受けて幸運にも合格しましたが、当時の理解は非常に浅かったと思います。秋からRAなどをやらせてもらいながら、大学院に向けた準備として以下の本を少しずつ読みました。

* 松坂和夫『集合・位相入門』
* 浅野・中村『計量経済学』
* MWG『Microeconomic Theory』
* D.Romer 『上級マクロ経済学』

これらは翌年一緒に大学院に入学する学生たちと一緒に、勉強会形式で読みました。しかし結局、あまり理解できないまま大学院の授業（いわゆるコースワーク）に突入することになりました。

## コースワーク
コースワークは大学院の1年次にあり、経済学の基礎固めとPh.D candidateの選抜を兼ねるものとなります。ミクロは前期に価格理論、後期にゲーム理論を勉強します。マクロは前期に代表的な通時的最適化問題（成長理論・貯蓄・労働・投資など）、後期に金融を勉強します。それぞれ授業は週1コマと、TAによる補習が2週間に1コマ程度あります。宿題も出ますが、評価はほとんど中間・期末試験で決まります。計量は後期のみで授業を週2コマやります。中心極限定理・大数の法則から始まって漸近理論をやります。重回帰分析・操作変数・GMM・時系列分析などを使えるようになります。先生にもよりますが、ノンパラメトリック・セミパラメトリックと呼ばれる、モデルの形を事前に仮定しない推定法も教わりました。
