### よく使うコマンド

[C-]：ctrlを押しながら

[M-]：「alt」を押しながら、もしくは「esc」を押してから

* C-x C-f　ファイルを開く
* C-x k　ファイルを閉じる
* C-x C-s　ファイルのセーブ
* C-Space　現在のカーソル位置をマーク
* C-k (kill-line) 行のカーソル以降を削除し kill-ring へ
* C-w (kill-region) region の範囲を削除して kill-ring へ
* M-w (copy-region-as-kill) region の範囲を削除せず kill-ring へ
* C-y (yank) kill-ring の内容をカーソル位置にはき出す
* C_c C_q　リージョンをインデント
* C-v　次の画面に移動
* M-v　前の画面に移動
* C-x 3　左右に 2分割
* C-x 2　上下に 2分割
* C-x 0　現在カーソルのあるウィンドウを削除
* C-_　Undo
* Redo 他のコマンドを使用すると，Undoコマンドの連続を中断．中断するとそれまでのUndoコマンド自身が取り消し可能な普通のコマンドとみなされる．このようにしてC-f(一文字進む)などのそれほど重要な効果のないコマンドを入力して，取り消された変更を再生出来る．そのあとはさらにUndoコマンドを使用可能
